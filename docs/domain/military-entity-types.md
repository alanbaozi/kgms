# 军事领域实体类型定义

日期：2026-05-24

本文档是 LightRAG 抽取配置的本地可读副本，用于后续讨论 KGMS 前端展示、过滤、关系映射和本体演进。实际抽取配置仍以 `deploy/lightrag-server/prompts/entity_type/military.yml` 为准。

## 设计原则

- 实体类型使用英文内部标识，前端显示中文名称。
- 内部标识使用 `snake_case`，与 LightRAG 最终存储的小写类型保持一致。
- 文档无论中文或英文，都映射到同一套实体类型。
- 国家、区域、地点、设施分开建模，避免把国家全部放进 `location`。
- 装备属性尽量通过关系连接到 `indicator`、`time`、`person`、`organization`、`force_unit`，不要只写进 description。
- 事件作为发生事项的容器，动作作为事件内或任务中的具体行为。

## 实体类型

| 内部类型 | 中文显示名 | 定义 | 示例 |
| --- | --- | --- | --- |
| `country` | 国家 | 主权国家、准国家政治实体或国家行为体。即使文本中作为地点出现，也优先归为国家。 | 中国、美国、伊朗、俄罗斯、俄国 |
| `force_unit` | 部队/军事力量 | 军种、战区、舰队、基地部队、航空兵旅、合成旅、编队、分队、任务编组、指挥所、火力单元。 | 中国海军、第72合成旅、第3航空兵编队 |
| `organization` | 组织机构 | 非部队组织，包括政府部门、军工企业、研究所、国际组织、媒体、智库、承包商。 | 北方航空工业集团、渤海造船重工业公司、某研究机构 |
| `person` | 人员 | 指挥员、官员、设计师、总师、专家、发言人、作者、观察员。 | 总设计师李明、某司令员、观察员 |
| `equipment` | 装备 | 武器、平台、系统、传感器、弹药、无人系统、舰艇、飞机、车辆、雷达、导弹、发动机、声纳，也包括装备类别。 | 歼-16、运油-20、09III型核潜艇、鹰击-18导弹、战斗机 |
| `facility` | 设施 | 军港、机场、空军基地、海军基地、仓库、雷达站、阵地、训练场、船厂、指挥设施、发射阵地。 | 阿巴斯港海军基地、某雷达站、某船厂 |
| `location` | 地点 | 国家以下或非政治实体的具体地理位置。 | 霍尔木兹海峡、某海域、某空域、某岛礁、坐标点 |
| `region` | 区域 | 跨国家或大范围区域、方向或作战区域。 | 中东、印太、西太平洋、南海、东海方向 |
| `event` | 事件 | 具有时间、地点、参与方、结果、公开记录或明确上下文的发生事项。 | 编队空中加油训练、海上演习、导弹试射、服役仪式 |
| `action` | 动作 | 事件中或任务中由明确主体执行的具体动作。必须能连接到主体、对象、装备、地点、时间或所属事件。 | 运油-20为歼-16实施空中加油、歼-16编队执行远海巡航 |
| `capability` | 能力 | 能力、作战用途或功能。 | 制空、反潜、远程打击、电子战、空中加油、预警探测 |
| `indicator` | 指标/属性值 | 可作为属性值检索的指标或状态。 | 2.1马赫、18000米、8吨、1500公里、舷号、风险等级 |
| `resource` | 资源 | 兵力、弹药、燃料、补给、经费、频谱、情报资料、后勤资源、备件。 | 燃料、弹药、补给、情报资料 |
| `time` | 时间 | 明确时间、日期、年份、阶段、窗口期、持续时间。 | 2012年、2025年6月2日、首飞时间、服役时间 |
| `plan` | 计划/方案 | 作战计划、任务方案、演训方案、应急预案、行动阶段、部署计划、条令方案。 | 演训方案、部署计划、应急预案 |
| `document` | 文档/资料 | 报告、命令、通报、条令、新闻稿、网页、表格、图像、地图、附件等资料来源。 | SIPRI 年鉴、新闻稿、装备表格 |
| `other` | 其他 | 无法归入上述类别但对检索有价值的概念或对象。 | 其他关键概念 |

## 边界规则

国家、区域、地点、设施：

- 伊朗、俄国、俄罗斯、美国、中国等国家必须归为 `country`，不要归为 `location`。
- 中东、印太、南海、西太平洋、东海方向等大范围区域归为 `region`。
- 霍尔木兹海峡、某海域、某空域、某岛礁、某坐标点归为 `location`。
- 赫梅米姆空军基地、阿巴斯港海军基地、某雷达站、某船厂归为 `facility`。

装备属性：

- 装备平台本体归为 `equipment`，例如歼-16、F-16、09III型核潜艇、鹰击-18导弹、运油-20加油机。
- 装备类别也归为 `equipment`，例如战斗机、加油机、核潜艇、反舰导弹、相控阵雷达。
- 载弹量、最大航速、巡航速度、实用升限、最高飞行高度、航程、作战半径、射程、探测距离、排水量、最大起飞重量等属性值归为 `indicator`。
- 设计时间、首飞时间、服役时间、退役时间归为 `time`。
- 设计师、总设计师归为 `person`；设计单位、制造单位归为 `organization`；服役部队归为 `force_unit`。

事件和动作：

- `event` 是发生事项的容器，通常连接时间、地点、参与方、参与装备、结果和来源。
- `action` 是事件内部或任务中的具体动作，必须连接执行主体、动作对象、使用装备、地点、时间或所属事件。
- 静态事实不要默认抽成 `action` 或 `event`，例如“服役于”“可搭载”“具备能力”“由某单位设计”。这些优先用关系表达。
- 只有出现明确发生语境时才抽 `event`，例如“某日举行演习”“完成试验”“发生事故”“组织编队空中加油训练”。
- 不要把低价值表达抽成 `action`，例如“显示”“指出”“包括”“认为”“提到”“报道”“称为”。
